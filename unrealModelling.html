Unreal Modellering/Rendering

Unreal

The project

This project was the first time i had to use unreal which was fun and the task was to modell house parts that could be used to build different buildings and after that build a world in unreal to put them all in. so as you might see is that all the houses are build with the same parts even though they look different.

The things i learned during this project was basic modelling in Maya, how the unreal engine works with lighting, i also learned the built in material editors and vertexpaint system inside unreal engine

here you can see the vertexpaint in action on the stone wall of the building where we blended moss with the base stone texture

To create the vertex paint we had to learn basic blueprint functionality to connect the different nodes for the vetex functionality
