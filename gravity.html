Gravity

The game

My role and the game in general

The game Gravity is a game i created along with a few classmates during a course at Stockholms Univeristy It is a 3D platformer that challenges what you thaught a 3D platformer can do by utilising all the possibilities of changeable gravity and three dimensions

The goal of the game is like any other platformer game, to reach the goal without failing.

The game is created in Unity and my role during the project was along with one classmate to code all the functionality in the game. what made this hard was that a part of the course was that we where not allowed to use any prebuilt scripts for collisions or character controlling so all the physics and everything related to it is coded by me and my classmate
